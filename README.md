# CircleCI AWS EC2 deployment example
Example of deploying a Node.js application to AWS EC2 using CircleCI

# Requirements
Before you start this tutorial, I assume you have the following:

* A Github repository
* A CircleCI account
* An AWS account
* A running AWS EC2 instance (AWS AMI)
